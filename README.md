# 프로젝트: 기독교 웹 서비스
- 가장 비슷한 신앙을 가진 성경 인물을 알려주는 서비스

### 목적
1. 기독교 관련 웹 서비스를 기획하고 배포하는 것
2. DaisyUI, HeadlessUI, Terraform 등 사용하고 싶었던 tech들을 사용하는 것.
3. 알맞는 CSR, SSR 적용 방법을 공부하는 것

### Tech Stack
- NextJS: Main Framework
- DaisyUI (TailwindCSS), HeadlessUI: Styling
